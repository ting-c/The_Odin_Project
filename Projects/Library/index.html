This library app allows you to add new books and remove existing books.


LIBRARY
  • Click on the Library tab to look at the existing book.
  • You can remove book or toggle the read status using the buttons at the bottom of each book.

ADD BOOKS
  • Click on the Add Books tab to add new books to the library.
  • You need to fill in all details before adding a new book.
